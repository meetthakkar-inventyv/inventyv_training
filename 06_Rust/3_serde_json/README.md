# Serde JSON (Rust)

This directory contains examples demonstrating **Serde with JSON** in Rust.

---

## folder Structure
```
3_serde_json/
├── 3_1_serde_serialization/
│ ├── src/
│ │ └── main.rs
│ ├── Cargo.toml
│ └── README.md
│
├── 3_2_serde_deserialization/
│ ├── src/
│ │ └── main.rs
│ ├── Cargo.toml
│ └── README.md
│
└── README.md```



---

## Overview

- **Serialization**: Rust ➜ JSON using `serde::Serialize`
- **Deserialization**: JSON ➜ Rust using `serde::Deserialize`


Each folder is a standalone Cargo project with its own example and README.
