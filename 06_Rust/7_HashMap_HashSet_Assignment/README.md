# HashMap & HashSet Assignment

This project demonstrates the practical usage of **HashMap** and **HashSet** in Rust, focusing on ownership, memory handling, and common collection methods.

## Folder Structure

```text
src/
├── main.rs
└── modules/
    ├── mod.rs
    ├── hashmap.rs
    └── hashset.rs

Cargo.toml
Cargo.lock
README.md
 ```

---


## Concepts Covered

- Creating and using `HashMap` and `HashSet`
- Inserting and managing data
- Ownership transfer using `std::mem::take`
- Cloning collections
- Filtering data using `retain`
- Extending collections using `extend`
- Understanding the `Default` trait behavior
