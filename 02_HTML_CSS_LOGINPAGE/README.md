# Assignment: HTML & CSS Login / Registration Page

This module contains a responsive login and registration webpage built using **plain HTML and CSS**, without any JavaScript or external frameworks.

---

## Files Included
- `index.html` – Page structure with Login and Register forms
- `style.css` – Styling for layout, cards, and responsiveness
- `logo.png` – Branding logo used in the UI

---

## Features
- Dual forms (Login & Register) switched using tabs/radio buttons
- Split-screen layout:
  - Left section (70%) for branding
  - Right section (30%) for authentication forms
- Modern card-style UI with blue accent colors
- Fully responsive design:
  - Stacks vertically on smaller screens
  - Mobile-friendly layout

---

## Objective
To practice HTML structure, CSS layouts, responsiveness, and clean UI design principles.
