# DBMS – Movie Database Queries

This module contains a MySQL assignment focused on querying a **Movie Database**, progressing from basic SQL commands to advanced queries.

---

## Files Included
- `movies_db_queries.sql` – SQL script containing all queries
- `sql_queries.md` – Documentation with questions, queries, and sample outputs
- `Questions.txt` – List of 50 problem statements grouped by topic

---

## Topics Covered
- Basic `SELECT`, filtering, and sorting
- Aggregate functions:
  - `COUNT`, `AVG`, `MIN`, `MAX`
- `GROUP BY` and `HAVING`
- Subqueries using `IN`, `EXISTS`, and correlated queries
- Joins between multiple tables:
  - Movies
  - Actors
  - Directors
  - Genres
  - Reviewers

---

## Objective
To gain hands-on experience with real-world SQL querying and understand relational database concepts in depth.
