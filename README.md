## Module Overview

### 01_LOGIC_BUILDING
This module focuses on logic building and algorithmic thinking.

- Created flowcharts for various patterns and algorithms
- Flowcharts are designed using Figma
- Emphasis on step-by-step problem-solving approaches

Contains a `README.md` with the Figma flowchart link.

---

### 02_HTML_CSS_LOGINPAGE
This module focuses on frontend development using only HTML and CSS.

- Developed a responsive Login/Register webpage
- No JavaScript or external frameworks used
- Focus on layout design, responsiveness, and clean UI

Files included:
- `index.html` – Page structure  
- `style.css` – Styling and responsiveness  
- `logo.png` – Logo asset  

---

###  03_DBMS
This module focuses on database management and SQL querying.

- Implemented 50 SQL queries on a Movie Database
- Queries include joins, subqueries, aggregation, and filtering
- Includes problem statements, queries, and outputs

Files included:
- `movies_db_queries.sql` – SQL queries  
- `sql_queries.md` – Queries with outputs  
- `Questions.txt` – Problem statements
