 ## Modules Overview

### 01_LOGIC_BUILDING
Focuses on developing logical thinking and problem-solving skills through structured approaches and flowcharts.

---

### 02_HTML_CSS_LOGINPAGE
A responsive login and registration page created using pure HTML and CSS, emphasizing layout design and UI fundamentals.

---

### 03_DBMS
Hands-on MySQL assignment involving multiple SQL queries on a movie database, covering essential DBMS concepts.

---

### 05_Exercism_Javascript

Completed the Exercism JavaScript module, solving various challenges to master language syntax, built-in methods, and logical implementation.
